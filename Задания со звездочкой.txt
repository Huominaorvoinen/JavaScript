*Чему будет равно JS-выражение 1000 + "108"?
Ответ: При выводе будет значение 1000108
*Самостоятельно разобраться с атрибутами тега script (async и defer).
Ответ: Эти атрибуты задают различные способы прогрузки для скрипта. Необходимость создается в основном для больших тегов, прогрузка которых может замедлить прогрузку самой страницы, если тег скрипта находится перед тегами HTML. При наличии атрибута defer прогрузка скрипта начинается в фоновом режиме вместе с загрузкой страницы, но выполняется по порядку. При наличии атрибутов async скрипты запускаются по мере прогрузки.
